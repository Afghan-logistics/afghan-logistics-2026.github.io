核心运输方式与专属服务
1. 海卡联运（中国→阿富汗）
核心路线：中国港口→伊朗阿巴斯 / 恰巴哈→阿富汗伊斯兰默卡拉口岸 / 米拉克口岸→赫拉特、坎大哈、喀布尔、马扎里沙里夫
重要备注：原经巴基斯坦杰曼、托克汉姆口岸的运输路线，因边境局势影响，自 2024 年起暂停运营，敬请留意。
2. 公路卡车运输（中国→阿富汗）
核心出境口岸：伊尔克什坦、霍尔果斯，分北疆 / 南疆双路线，精准适配不同货物运输需求
北疆路线：国内装货→霍尔果斯仓库换装→霍尔果斯口岸出境→哈萨克斯坦、乌兹别克斯坦→阿富汗海拉顿
✅ 核心优势：一次换装、路况平坦，可承接大件设备 / 特种物资运输
南疆路线：国内装货→乌恰仓库换装→伊尔克斯坦口岸出境→吉尔吉斯换装→乌兹别克斯坦→阿富汗海拉顿
✅ 核心优势：运营车辆充足、运价稳定，适配普货 / 轻泡货运输
适用货物：大件设备、轻泡货、超长 / 超宽 / 超高 / 超重特种物资

3. 铁路班列运输（中国→阿富汗海拉顿）
固定路线：国内起运车站→霍尔果斯口岸出境→哈萨克斯坦→乌兹别克斯坦→阿富汗海拉顿车站
运输时效：18-25 天
发运条件：需提前申请运输计划，集齐 42 个整柜统一发车
4. 公铁联运・中阿快线（中国→阿富汗）
2023 年正式独家开通，为当前中国至阿富汗最主流、最安全的运输路线，融合公路 + 铁路双重优势，在伊朗局势背景下更显运输可靠性：
核心路线：国内起运火车站→喀什车站→汽运至伊尔斯坦口岸→吉尔吉斯奥什车站→铁路经塔什干→阿富汗海拉顿车站
核心优势：随装随发、一箱一发、调度灵活、运输高效
支持箱型：40GP、40HQ、45HQ
运输时效：门到站全程约 20-25 天
